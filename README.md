# Todo App 

## Description

A task management application where users can add, delete, and mark tasks as completed.

## Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* Data stored using localStorage

## Tech Stack

* HTML
* CSS
* JavaScript
