# Weather App 🌤️

##  Description

A weather application that fetches real-time weather data using an API.

##  Features

* Search weather by city name
* Displays temperature and weather condition
* Error handling for invalid input

## Tech Stack

* HTML
* CSS
* JavaScript
* Weather API
